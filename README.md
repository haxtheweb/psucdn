# PSU web components CDN
This is a deployment of built CDN assets for a pure CDN and HAXCMS based CDN

```html
<script>
  window.__appCDN = "https://cdn.hax.cloud/cdn/";
</script>
<script src="https://cdn.hax.cloud/cdn/build.js"></script>
```
